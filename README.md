# chemistry-prediction

基于 PyTorch 的化学溶解度预测项目。任务是利用 7 个分子描述符预测有机分子的水溶解度 `logS`，属于回归问题。

## 项目内容

本仓库包含：

- 数据探索与可视化
- 训练集 / 验证集 / 测试集划分
- 标准化预处理
- 线性基线模型
- PyTorch MLP 模型
- 5-fold 交叉验证
- 二阶特征扩展与增强模型
- 多 seed 集成评估
- 误差分析与特征重要性分析

## 主要文件

- `chemical prediction.ipynb`：主 Notebook，包含完整 PyTorch 训练、调参、评估与集成流程
- `chemical_solubility_data.csv`：化学溶解度数据集
- `化学溶解度预测_第一组.ipynb`：另一份 Notebook 版本
- `课程项目二(1).ipynb`：课程项目原始说明与任务要求
- `课程项目大模型使用指南(1).ipynb`：课程中关于 LLM 使用的说明
- `LLM交互记录.md`：我和 LLM 的交互过程整理

## 当前效果

目前 Notebook 中的最佳结果来自 5 模型集成：

- Test RMSE: `0.5600`
- Test MAE: `0.4224`
- Test R2: `0.9754`

相较于原始 MLP，集成模型进一步降低了误差，并且结果更稳定。

## 运行方式

1. 打开 `chemical prediction.ipynb`
2. 选择 Python / PyTorch 环境
3. 从上到下依次运行所有单元
4. 查看训练曲线、交叉验证结果、集成结果和误差分析表

## 建模思路

这份项目的思路是先用简单模型建立基线，再逐步提升：

1. 线性回归作为 baseline
2. MLP 作为非线性模型
3. 5-fold 交叉验证检查稳定性
4. 多项式特征 + 更深网络提升拟合能力
5. 多 seed 集成进一步降低泛化误差

## 和 LLM 的协作方式

在这个项目里，LLM 主要用于：

- 梳理课程项目目标
- 检查 Notebook 的执行顺序和潜在 bug
- 帮助设计更稳健的评估方案
- 解释误差分析和特征重要性
- 整理成可提交的 Markdown 记录
