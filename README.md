# chemistry-prediction

基于 PyTorch 的化学溶解度预测项目。任务是利用 7 个分子描述符预测有机分子的水溶解度 `logS`，属于回归问题。

## 项目简介

本项目围绕“从分子特征预测化学溶解度”展开，核心目标是搭建一个完整、可复现、可解释的回归建模流程。我们先从数据理解和基础探索入手，再逐步引入更强的模型与更稳健的验证方式，最后通过集成与误差分析进一步提升预测效果。

### 任务目标

- 理解 7 个分子描述符的化学意义
- 建立从数据探索到模型评估的完整流程
- 对比线性模型、MLP 与增强模型的表现
- 通过交叉验证、集成和特征分析提升泛化能力
- 让 Notebook 的结论更适合作业展示与课程汇报

## 数据说明

数据集 `chemical_solubility_data.csv` 包含 500 个样本，每个样本有 7 个输入特征和 1 个目标值 `logS`。

### 特征列表

- `MolWt`：分子量
- `LogP`：脂水分配相关特征
- `NumHDonors`：氢键供体数量
- `NumHAcceptors`：氢键受体数量
- `TPSA`：拓扑极性表面积
- `NumRotatableBonds`：可旋转键数量
- `AromaticProportion`：芳香性比例

### 目标变量

- `logS`：水溶解度的对数值

## 实验流程

Notebook 中的实验流程按以下顺序展开：

1. 数据加载与基本统计检查
2. 特征分布可视化
3. 相关性热力图分析
4. 训练集 / 验证集 / 测试集划分
5. 标准化预处理
6. 线性基线模型
7. PyTorch MLP 模型
8. 5-fold 交叉验证
9. 二阶特征扩展 + 增强 MLP
10. 多 seed 集成评估
11. 误差分析与特征重要性分析

## 当前结果

目前 Notebook 中表现最好的方案是 5 模型集成。最终测试集结果如下：

| 模型 | Test RMSE | Test MAE | Test R2 |
| --- | --- | --- | --- |
| 线性基线 | 0.7351 | - | 0.9562 |
| 原始 MLP | 0.6913 | - | 0.9613 |
| TunedMLP + Poly2 | 0.5865 | - | 0.9736 |
| 5 模型集成 | 0.5600 | 0.4224 | 0.9754 |

说明：

- 线性模型可以作为 baseline
- 增强 MLP 明显优于普通 MLP
- 集成模型当前是效果最好的版本，误差最低、稳定性最好

## 结果解读

从结果上看，模型主要学到了与溶解度关系更密切的化学信息，尤其是：

- 分子量 `MolWt`
- 疏水性 `LogP`
- 极性表面积 `TPSA`

置换特征重要性也表明，这些特征被模型认为是最关键的输入变量。

## 建模思路

本项目采用“先基线、再增强、最后集成”的策略：

1. 先用线性模型建立基线
2. 再用 MLP 学习非线性关系
3. 用 5-fold 交叉验证检查稳定性
4. 使用多项式特征和更深网络提升表达能力
5. 使用多 seed 集成进一步压低泛化误差
